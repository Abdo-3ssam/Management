# Management Dashboard (Power BI)

![Management Dashboard](https://github.com/Abdo-3ssam/Management/blob/main/Management-Dashboard.png)

## Overview
This **Management Dashboard** is built with **Power BI** to provide interactive and insightful visualizations of business metrics. It helps managers and stakeholders monitor revenue, expenses, income, property sales, and client performance in real-time.

## Key Features
- **Revenue Insights:** Analyze revenue by month, type, country, and sales channels.
- **Expense & Income Tracking:** Visual comparison of expenses vs income.
- **Property Sales Overview:** Track properties sold with detailed status (Paid, Pending).
- **Client Analytics:** Identify top clients and their contributions.
- **Interactive Visuals:** Rich charts, graphs, and geographic maps for intuitive data exploration.
- **Dynamic Filtering:** Easily filter data by year, location, and sales type.

## Dashboard Sections
1. **Revenue & Expenses Summary:** Quick snapshot of total revenue, expenses, and income.
2. **Revenue by Month:** Monthly trends in revenue.
3. **Revenue by Type & Sales Means:** Breakdown by property type and sales channel.
4. **Revenue by Country:** Geographic distribution of revenue.
5. **Property Status Table:** Detailed table of properties with sale date, price, and payment status.
6. **Best Clients:** Highlights top clients and their roles.

## How to Use
1. Open the Power BI Desktop file (`.pbix`) included in the repository.
2. Refresh the data to get the latest figures.
3. Use slicers to filter by year, location, or sales channel for deeper insights.

## Requirements
- [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
- Data source (Excel, CSV, or database) connected to the dashboard.
